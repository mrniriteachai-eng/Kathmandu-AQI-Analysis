**Nepal Air Quality Monitor** is a comprehensive environmental data science project that analyzes air quality in Kathmandu Valley. This project tracks AQI (Air Quality Index), PM2.5, PM10, NO2, SO2 levels, detects pollution trends, and predicts future air quality using Python, Pandas, NumPy, and Matplotlib.

The system processes 10+ days of air quality data, categorizes pollution levels, calculates moving averages, detects improving or worsening trends, and generates professional environmental dashboards.

## 02 WHAT IS AQI (AIR QUALITY INDEX)

<div align="center">

<img src="https://media.giphy.com/media/3o6Zt481isNVuQI1l6/giphy.gif" width="250">

</div>

**AQI (Air Quality Index)** is a number used by government agencies to communicate how polluted the air currently is or how polluted it is forecast to become.

### AQI CATEGORIES AND COLORS

| AQI RANGE | CATEGORY | COLOR | HEALTH IMPACT |
|-----------|----------|-------|---------------|
| 0 - 50 | Good | 🟢 Green | Air quality is satisfactory |
| 51 - 100 | Moderate | 🟡 Yellow | Acceptable air quality |
| 101 - 150 | Unhealthy for Sensitive | 🟠 Orange | Sensitive groups may have effects |
| 151 - 200 | Unhealthy | 🔴 Red | Everyone may begin to experience effects |
| 201 - 300 | Very Unhealthy | 🟣 Purple | Health alert: everyone may experience serious effects |
| 301 - 500 | Hazardous | ⚫ Maroon | Health warning of emergency conditions |

### KATHMANDU'S AQI PROBLEM

| SEASON | TYPICAL AQI | STATUS |
|--------|-------------|--------|
| Winter (Dec-Feb) | 150-200 | Unhealthy |
| Spring (Mar-May) | 120-180 | Unhealthy for Sensitive |
| Monsoon (Jun-Aug) | 50-100 | Moderate |
| Autumn (Sep-Nov) | 80-130 | Moderate to Unhealthy |

"""
NEPAL AIR QUALITY MONITOR - ADVANCED ANALYSIS
Nirmal Saud | AI Engineer | MrNiriTeach
"""

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

print("=" * 60)
print("🌿 NEPAL AIR QUALITY MONITOR - ADVANCED ANALYSIS")
print("=" * 60)

# Load data
df = pd.read_csv('../data/kathmandu_aqi.csv')

# AQI Category Function
def get_aqi_category(aqi):
    if aqi <= 50:
        return "Good"
    elif aqi <= 100:
        return "Moderate"
    elif aqi <= 150:
        return "Unhealthy (Sensitive)"
    elif aqi <= 200:
        return "Unhealthy"
    elif aqi <= 300:
        return "Very Unhealthy"
    else:
        return "Hazardous"

def get_aqi_color(aqi):
    if aqi <= 50:
        return '#1ccc1c'
    elif aqi <= 100:
        return '#ffff00'
    elif aqi <= 150:
        return '#ffa500'
    elif aqi <= 200:
        return '#ff0000'
    elif aqi <= 300:
        return '#800080'
    else:
        return '#800000'

# Apply categorization
df['Category'] = df['AQI'].apply(get_aqi_category)
df['Color'] = df['AQI'].apply(get_aqi_color)

# Category distribution
print("\n📊 AQI CATEGORY DISTRIBUTION:")
category_counts = df['Category'].value_counts()
for cat, count in category_counts.items():
    print(f"   {cat}: {count} days ({count/len(df)*100:.1f}%)")

# Pollutant analysis
print("\n💨 POLLUTANT LEVELS:")
print(f"   PM2.5 Average: {df['PM25'].mean():.2f} µg/m³")
print(f"   PM10 Average: {df['PM10'].mean():.2f} µg/m³")
print(f"   NO2 Average: {df['NO2'].mean():.2f} ppb")
print(f"   SO2 Average: {df['SO2'].mean():.2f} ppb")
